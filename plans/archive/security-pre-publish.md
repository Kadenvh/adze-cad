# Security Pre-Publish Plan

**Status:** Completed on 2026-03-13
**Priority:** P0
**Created:** 2026-03-13

---

## Context

The codebase has no hardcoded secrets — API keys are loaded from environment variables, and `.claude/hooks` blocks edits to credential files. However, repository hygiene issues must be resolved before the first commit to `adze-cad`.

Execution result:
- `.gitignore` was added and extended for local tooling/state.
- Generated `bin/`, `obj/`, logs, installer downloads, temp files, and benchmark report JSON were removed.
- Remaining diagnostic scripts were sanitized to avoid hardcoded user profile paths.
- `git add --dry-run .` no longer shows the sensitive patterns screened by this plan.

## Step 3: Sanitize benchmark reports

The following files contain machine-specific `machine_name` values:

```
benchmarks/reports/broker-evals-report-latest.json
benchmarks/reports/grounding-benchmarks-report-latest.json
benchmarks/reports/broker-evals-report-*.json  (timestamped copies)
benchmarks/reports/grounding-benchmarks-report-*.json  (timestamped copies)
```

**Options (choose one):**
- A. Delete all report JSON files. They can be regenerated by running the benchmark scripts.
- B. Replace the machine-specific `machine_name` value with `"machine_name": "dev"` in all report files.
- C. Keep `-latest.json` only (anonymized), delete timestamped history.

Recommended: Option A. Reports are generated artifacts.

## Step 4: Sanitize documentation references

Check these files for machine-specific info and replace if needed:

- `CLAUDE.md` — Check for any machine-specific validation wording and generalize if needed.
- `documentation/IMPLEMENTATION_PLAN.md` — May reference machine name in validation results.
- `documentation/PROJECT_ROADMAP.md` — Check for local-specific paths.
- `benchmarks/grounding/starter-corpus.manifest.json` — Contains local paths like `C:\SOLIDWORKS\samples\` and `C:\Program Files\Dassault Systemes\...`. These are fixture references, not secrets, but consider whether they should be in a public repo.
