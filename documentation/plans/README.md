# Plans

This directory contains supporting planning artifacts, discovery work, diagnostics, and handoff documents for Adze.

## Important Scope Rule

The canonical project record lives one directory up:

- `documentation/PROJECT_ROADMAP.md`
- `documentation/BUILD_SPEC.md`
- `documentation/IMPLEMENTATION_PLAN.md`
- `documentation/FIRST_USABLE_BUILD.md`

Use the files here for historical context, detailed discovery rationale, diagnostics, and agent handoff. Do not treat them as the primary product record when they conflict with the canonical docs.

## Contents

| File/Folder | Purpose |
|-------------|---------|
| `discovery-adze-for-solidworks.md` | Discovery brief and pre-bootstrap research record |
| `development-roadmap-handoff.md` | Agent-oriented phased handoff for current engineering work |
| `security-pre-publish.md` | Historical pre-publish cleanup plan and execution record |
| `rename-to-adze.md` | Historical rename plan and execution record |
| `diagnostics/` | Scriptable environment checks and planning-era diagnostics |

## Conventions

- Keep artifacts here factual and date-specific.
- Summarize decisions in the canonical docs instead of duplicating them forever here.
- Treat `diagnostics/` as support material, not product source.
