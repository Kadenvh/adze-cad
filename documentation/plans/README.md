# Plans

This directory contains discovery briefs, design documents, and the end-goal vision for Adze.

## Important Scope Rule

The canonical project record lives one directory up:

- `documentation/PROJECT_ROADMAP.md`
- `documentation/BUILD_SPEC.md`
- `documentation/IMPLEMENTATION_PLAN.md`
- `documentation/FIRST_USABLE_BUILD.md`

Files here are research, discovery, and planning artifacts. When they conflict with the canonical docs, the canonical docs win.
